# Christian Wilfredo — Portfolio

A personal portfolio website split into two worlds: **IT Projects** and **Life**.

## ✨ Features

- **Split-screen landing page** — hover to expand, click to enter
- **IT Projects** — showcasing web & IoT projects with animated cards
- **Life** — MUN club journey (member → president) and tennis passion
- **Elegant design** — warm monochromes, gold accents, Playfair Display typography
- **Scroll-reveal animations** — elements animate in as you scroll
- **Responsive** — works on desktop, tablet, and mobile
- **Social links** — GitHub, LinkedIn, Instagram, Email on every page

## 🛠️ Tech Stack

- HTML5
- CSS3 (vanilla, no frameworks)
- JavaScript (vanilla)
- [Lucide Icons](https://lucide.dev/) (CDN)
- [Google Fonts](https://fonts.google.com/) — Playfair Display & Inter

## 📁 Project Structure

```
├── index.html            # Split-screen landing page
├── projects.html         # IT Projects showcase
├── life.html             # MUN & Tennis sections
├── css/
│   └── style.css         # Design system & all styles
├── js/
│   └── main.js           # Interactions & animations
└── assets/
    └── images/           # Photos & generated images
```

## 🚀 Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/YOUR_USERNAME/portfolio.git
   ```
2. Open `index.html` in your browser — no build step needed!

## 📝 Customization

- **Projects**: Edit the project cards in `projects.html`
- **MUN/Tennis content**: Edit sections in `life.html`
- **Social links**: Update `href` attributes in all HTML files
- **Colors**: Modify CSS variables in `:root` inside `css/style.css`
- **Images**: Replace files in `assets/images/` with your own photos
